    第一篇博客——记录上传部署博客的过程

    第一：找文件夹

    找到blog的文件夹，这里放着配置文件，主题，文章…（地址在C:\Users\WangTao\blog）

    然后找到source文件夹，到里面的_posts文件夹，这里放着文章。把文章放里面。

    第二：在终端输命令

    终端，不是cmd，想快点就cmd这里找。

    cd 过 C:\Users\WangTao\blog 来

    hexo server 就可以在本地的4000端口看效果

    hexo generate 转成静态网页

    hexo deploy 就部署，搞定

    遇到的问题

    1. 报错
      hexo : 无法加载文件 C:\Users\WangTao\AppData\Roaming\npm\hexo.ps1，因为在此系统上禁止运行脚本。
      解决方法：

      1. 管理员 PowerShall

      2. 输Set-ExecutionPolicy RemoteSigned改变执行策略，询问是否时选Y

    2. git’ 不是内部或外部命令，也不是可运行的程序或批处理文件。
      git没放在环境变量里
      解决方法：找到git (git bash),增加环境变量。
