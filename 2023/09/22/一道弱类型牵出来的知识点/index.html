    一道弱类型牵出来的知识点

    题目

    一道php弱类型题，首先应该看懂正则，搞懂哪些限制；然后知道preg_match()的绕过方式；最后知道使用函数查看文件获得flag

    正则

    正则表达式是什么？

    PHP正则表达式是形成搜索模式的字符序列，我们可以利用它实现对传入字符串的查询搜索（发现危险的东东就删掉或换掉），或是替换

    在php中其是   由 分隔符、模式 和 可选择修饰符 组成的字符串

    eg：/ 模式 / 修饰符

    注：分隔符可以是任何非字母，数字，\ 或 空格的字符  / # ~… 都行

    函数语法

    使用的函数

    preg_match(string $pattern, string $subject  […] )     match(匹配模式)   ——》在字符串中找到该模式，成功返回1，否则0；  $pattern是正则表达式， $subject是目标字符串；

    preg_match_all()    （匹配所有模式）——》 在字符串中找到匹配该模式的次数，返回次数

    preg_replace()    replace(代替)  ——》会将匹配模式中的字符换掉，返回替换后的字符串

    语法

    • 字面值字符：字母，数字，空格等，可以直接匹配

    • 特殊字符：eg：* + ？等，具有特殊含义和功能

      • *：匹配前面的子表达式0次或多次
        啥叫子表达式？–》eg:abcd* 这里的d就是“前面的子表达式”，匹配的时候会匹配abc+零次或多次d 相当于{0,}

      • + ： 匹配一次或多次

      • . ： 匹配 \n 以外的任何字符

      • ? ：匹配零次或一次

    • 字符类：用方括号 [ ] 包围的字符集合，用于匹配方括号内的任意一个字符。
      eg：[abcd] 就能匹配abcd中的任一字符，它会看作为一个整体，可以在后面接后续操作
      一个正则表达式遍历搜索的时候是从头搜索起的，直到遇到一个不符合的

      这个例子会遍历搜索表达式里有abcd的地方
      但是遇到第一个不符合的地方时就会“断掉”

      这里就会在b这个地方断掉

    • 元字符：例如 \d、\w、\s 等，用于匹配特定类型的字符，如数字、字母、空白字符等。

    • 修饰符：i g m s ...

    • 量词：例如 {n}、{n,}、{n,m} 等，用于指定匹配的次数或范围。

    • 边界符号：例如 ^、$、\b、\B 等，用于匹配字符串的开头、结尾或单词边界位置。

      • ^表示字符 开头

      • $表示 结束
        具体的遇到就百度吧

    变量字符的绕过

    （前面获取变量的地方没截下来）我们看到他要验证k_e_y传参的值，但是参数不能输入_，就考到非法字符替换

    简单说来，php在获取变量之后会将他们转化为有效变量名，此时就会替换非法字符

    eg： [     &  空格  +  .   = 以及他们的url编码

    所以面对检查问题时我们往往通过此方法绕过

    实测 + 可绕过

    2. PCRE回溯次数限制（什么归东东，不太懂）
    3. 换行符
    4. 尾句%0a绕过

    php函数实现无参数读取文件

    在这里我们应该首先想到文件读取，但是过滤了很多字符，发现括号是中文的。那么就是提示我们使用函数

    要使用文件读取

    过滤了那么多，就只能用无参数文件读取

    eval函数使得存在RCE（远程代码执行漏洞），我们利用它实现我们的函数执行

    首先我们应该知道文件夹名

    正常的情况，print_r(scandir('.')); 可以看到当前文件夹名，但是.是被过滤的。所以我们应该制造这个点

    制造点

    1. localeconv()

      localeconv()返回一包含本地数字及货币格式信息的数组。而数组第一项就是"."

      此时我们的需求就是获取第一个字符

      那么就要献出

      current()返回数组中的单元，默认取第一个值

      reset()函数

      所以方法=print_r(scandir(pos(localeconv())));其中pos是current别名

      print_r(scandir(current(localeconv())));

    2. chr(46)

      chr(46) = .

    3. phpversion()

      通过取数函数运算的方式得到46

    读文件

    知道了文件夹名，而后要读文件

    读文件就是文件包含漏洞

    经典函数：

    • readfile

    • highlight_file

    • file_get_contents

    查看上一级目录文件名

    • dirname()返回路径中目录的部分

    • 构造.. —— 就是两个点

      print_r(next(scandir(getcwd())));

    读取上级目录文件

    show_source(array_rand(array_flip(scandir(dirname(chdir(dirname(getcwd())))))));
